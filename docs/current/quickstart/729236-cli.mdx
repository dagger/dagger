# Quickstart

## Install the Dagger CLI

The next step is to install the Dagger CLI. Run the following commands:

```shell
cd /usr/local
curl -L https://dl.dagger.io/dagger/install.sh | sh
```

The above command will create `/usr/local/bin/dagger`.
