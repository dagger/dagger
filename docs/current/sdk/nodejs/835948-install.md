# Installation

:::note
The Dagger Node.js SDK requires [Node.js 16.x or later](https://nodejs.org/en/download/).
:::

Install the Dagger Node.js SDK in your project using `npm` or `yarn`:

<Tabs>
<TabItem value="npm">

```shell
npm install @dagger.io/dagger
```

</TabItem>

<TabItem value="yarn">

```shell
yarn add @dagger.io/dagger
```

</TabItem>
</Tabs>
