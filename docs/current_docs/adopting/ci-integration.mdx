# CI Integration

Dagger runs in any CI system that supports containers. The integration is a single command:

```yaml
# GitHub Actions
- run: dagger check

# GitLab CI
script:
  - dagger check

# Any CI
steps:
  - run: dagger check
```

Because Dagger runs identically everywhere, a passing `dagger check` locally guarantees the same result in CI.

## Cloud Checks

For GitHub repositories, [Cloud Checks](https://dagger.cloud) can run your checks automatically on every push — no CI configuration needed:

1. `dagger login`
2. Connect your repository in the Dagger Cloud dashboard
3. Push code — checks run automatically and report as commit statuses
