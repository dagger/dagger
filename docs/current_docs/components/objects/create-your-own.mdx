---
title: "Create Your Own"
description: "Create your own Component as a Dagger Module to extend the functionality of Dagger."
---

In addition to the default Dagger Components, you can create and add your own custom Components to Dagger. These custom Components can be used in Dagger Modules and can be composed with other core Components to create complex workflows.

Learn more about creating custom Components in [Extending Dagger](../../extending/index.mdx).
