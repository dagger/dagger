description: "Learn how to extend Dagger by embedding it into your own applications, enabling custom workflows and integrations."
---

In addition to using the Dagger SDKs, you can also embed Dagger into your own applications. This allows you to create custom applications that can use Dagger's capabilities, such as building and running containers, managing secrets, and more.
