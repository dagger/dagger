# Java SDK Docs Outline

The Java SDK docs should teach Java developers how to work with Dagger through Maven, generated sources, annotations, and Java's type system.

## Proposed Structure

1. Start with a Java module.
2. Understand Maven project layout and generated sources.
3. Define module classes, functions, arguments, and return values.
4. Document APIs with Java annotations and comments.
5. Work with core Dagger types.
6. Add module dependencies.
7. Use default paths, ignore patterns, checks, and generators.
8. Test Java modules.
9. Configure IDE support.
10. Troubleshoot Maven, annotation, and generated-source failures.
11. Release and maintain Java modules.
