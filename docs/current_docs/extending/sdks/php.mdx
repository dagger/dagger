# PHP SDK Docs Outline

The PHP SDK docs should teach PHP developers how to build Dagger modules with Composer, generated SDK packages, PHP attributes, and editor support.

## Proposed Structure

1. Start with a PHP module.
2. Understand Composer layout and the generated `dagger/dagger` package.
3. Define module objects and functions.
4. Use PHP types, attributes, and documentation.
5. Work with core Dagger types.
6. Add module dependencies.
7. Use default paths, ignore patterns, checks, and generators.
8. Test PHP modules.
9. Configure PhpStorm, VS Code, and language servers.
10. Troubleshoot Composer and generated-package failures.
11. Release and maintain PHP modules.
