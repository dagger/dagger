# What is Dagger?

Dagger is a platform for automating software delivery. It can build, test and ship any codebase, reliably and at scale.

Dagger runs locally, in your CI server, or directly in the cloud.

## Why Dagger?

**Programmable.** Write delivery workflows in code, not YAML. SDKs for Go, Python, TypeScript, PHP, and more.

**Local-first.** Runs anywhere with a container runtime. Your laptop, CI server, or cloud infrastructure — same behavior everywhere.

**Repeatable.** Everything runs in containers. Every operation is cached by its inputs. Same inputs, same outputs.

**Observable.** Every operation emits OpenTelemetry traces. Debug in the terminal TUI or in a web view.

[Dagger Cloud](https://dagger.cloud) adds managed infrastructure: run checks on every push with Cloud Checks, visualize traces across your organization, and manage modules centrally.
