# Core Concepts

Dagger is built around a few core concepts that work together to enable powerful, reusable automation workflows.

---

## [Workspaces](/core-concepts/workspaces)

**Your project's Dagger configuration.**

A workspace is a directory with a `.dagger/` folder that defines which modules your project uses and how they're configured. It's the connection between your codebase and Dagger.

```shell
# Initialize a workspace
dagger init

# Install a module
dagger install github.com/dagger/eslint
```

[Learn about Workspaces →](/core-concepts/workspaces)

---

## [Modules](/core-concepts/modules)

**Packages of functions you install or develop.**

Modules add capabilities to your workspace. Install pre-built modules from the community, or write your own. Each module provides functions you can call and checks you can run.

```shell
# Install a module
dagger install github.com/dagger/vitest

# See its functions
dagger call vitest --help
```

[Learn about Modules →](/core-concepts/modules)

---

## [Functions](/core-concepts/functions)

**The building blocks you call.**

Functions are units of computation that accept inputs, perform operations, and return outputs. When you run `dagger call`, you're calling a function.

```shell
# Call a function
dagger call eslint lint

# Chain functions
dagger call vitest test
```

[Learn about Functions →](/core-concepts/functions)

---

## [Checks](/core-concepts/checks)

**Functions that validate with no required arguments.**

Checks are special functions that validate your code and run with a single command. Every installed module can provide checks that run concurrently.

```shell
# Run all checks
dagger check

# Run specific checks
dagger check lint-*
```

[Learn about Checks →](/core-concepts/checks)

---

## How They Work Together

1. **Workspaces** declare which **Modules** your project uses
2. **Modules** provide **Functions** and **Checks**
3. **Functions** are what you call with `dagger call`
4. **Checks** are functions that validate, run with `dagger check`

### Example

```shell
# Set up your workspace with modules
dagger install github.com/dagger/eslint
dagger install github.com/dagger/vitest
dagger install github.com/dagger/prettier

# Use their functions
dagger call eslint lint
dagger call vitest test

# Run all checks at once
dagger check
```
