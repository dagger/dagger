# Administrator Manual

Welcome to the Dagger administrator manual!

This manual is intended to help Dagger administrators and operators use Dagger in production environments.

It includes information on available Dagger [integrations](../../integrations/index.mdx), including for popular CI providers/tools like [GitHub Actions](../../integrations/642920-github-actions.mdx), [GitLab CI](../../integrations/734201-gitlab.mdx), and [Jenkins](../../integrations/322019-jenkins.mdx).

It also provides information on using Dagger with [Kubernetes](../../integrations/104820-kubernetes.mdx), and [Dagger Cloud administration](./cloud/index.mdx).
