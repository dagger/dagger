:::note
Enumerations will be displayed when calling `--help` on a Dagger Function.
:::

Here's an example of displaying a Dagger Function helper on this method:

```
dagger call scan --help
```

The result will be:

```
USAGE
  dagger call scan [arguments]

ARGUMENTS
      --ref string                                  [required]
      --severity UNKNOWN,LOW,MEDIUM,HIGH,CRITICAL   [required]
```

Here's an example of calling the Dagger Function with an invalid enum argument, which will result in an error that displays
possible values:

```
dagger call scan --ref=hello-world:latest --severity=FOO
```

The result will be:

```
Error: invalid argument "FOO" for "--severity" flag: value should be one of UNKNOWN,LOW,MEDIUM,HIGH,CRITICAL
Run 'dagger call scan --help' for usage.
```
