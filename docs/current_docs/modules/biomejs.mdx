# Biome

Official module: [dagger/biomejs](https://github.com/dagger/biomejs)

Install with `dagger mod install github.com/dagger/biomejs`.

## What It Is

The Biome module lints source and can fix issues by returning a changeset.

## Use It For

- Formatting JS and TS projects
- Running Biome lint checks
- Replacing or complementing ESLint and Prettier in a workspace

## Functions

| Function | Purpose |
| --- | --- |
| `lint` | Lint the source code. |
| `fix` | Fix lint issues and return a changeset of the result. |

## How to Read This API

Use `lint` for validation. Use `fix` when the workflow should propose repaired source.

## Checks

Use `lint` as the likely workspace check.

## Settings

Document source discovery, config path, Biome version, args, and how `fix` returns changes.
