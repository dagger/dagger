# ESLint

Official module: [dagger/eslint](https://github.com/dagger/eslint)

Install with `dagger mod install github.com/dagger/eslint`.

## What It Is

The ESLint module lints JavaScript and TypeScript source. It uses a Node base image, a package manager, and a project source directory.

## Use It For

- Linting JS and TS source
- Adding lint checks to a workspace
- Sharing ESLint defaults across a team

## Functions

| Function | Purpose |
| --- | --- |
| `lint` | Lint the source. |
| `fix` | Fix linting issues. |
| `source` | Source directory for the project. |
| `package-manager` | Package manager to use. |
| `base-image-address` | Base image to use. |

## How to Read This API

Use `lint` for validation. Use `fix` when the workflow should return corrected source or a changeset.
