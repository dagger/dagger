# Helm

Official module: [dagger/helm](https://github.com/dagger/helm)

Install with `dagger mod install github.com/dagger/helm`.

## What It Is

The Helm module discovers Helm charts in a workspace, lints them, and checks that values files render with `helm template`.

## Use It For

- Linting Helm charts
- Rendering templates
- Packaging charts
- Validating chart changes in CI

## Functions

| Function | Purpose |
| --- | --- |
| `lint` | Lint every discovered Helm chart in the workspace. |
| `assert-template` | Assert every discovered values file renders with `helm template --dry-run=client`. |
| `charts` | Scan the workspace for Helm charts. |
| `chart` | Return one discovered chart by workspace-relative path. |
| `values-glob` | Glob for chart values files to check as extra matrix dimensions. |
| `version` | Helm version to use. |

## How to Read This API

Use `lint` for chart validation. Use `assert-template` when values files should be rendered as part of the check.

Use `charts` and `chart` for workflows that need to inspect chart discovery or operate on one chart.

## Checks

Use `lint` and `assert-template` as the likely workspace checks.
