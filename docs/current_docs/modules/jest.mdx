# Jest

Official module: [dagger/jest](https://github.com/dagger/jest)

Install with `dagger mod install github.com/dagger/jest`.

## What It Is

The Jest module runs Jest tests for JavaScript and TypeScript workspaces. It uses a Node base image, a package manager, and a project source directory.

## Use It For

- Running Jest tests
- Adding a test check to a Node workspace
- Sharing test settings across a team

## Functions

| Function | Purpose |
| --- | --- |
| `test` | Execute the tests. |
| `list` | List the tests. |
| `source` | Source directory for the project. |
| `package-manager` | Package manager to use. |
| `base-image-address` | Base image to use. |

## How to Read This API

Use `test` for normal validation. Use `list` when a workflow needs to inspect test discovery without running the suite.
