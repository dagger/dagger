# Prettier

Official module: [dagger/prettier](https://github.com/dagger/prettier)

Install with `dagger mod install github.com/dagger/prettier`.

## What It Is

The Prettier module checks and rewrites formatting for files handled by Prettier. It uses a Node base image, package manager, and project source directory.

## Use It For

- Checking formatting
- Producing formatting changesets
- Sharing formatting defaults across a workspace

## Functions

| Function | Purpose |
| --- | --- |
| `check` | Check whether files are formatted. |
| `write` | Rewrite processed files in place. |
| `source` | Source directory for the project. |
| `package-manager` | Package manager to use. |
| `base-image-address` | Base image to use. |

## How to Read This API

Use `check` for CI and Cloud. Use `write` for a formatting changeset or local repair workflow.

## Checks

Use `check` as the likely workspace check.

## Settings

Document `source`, config path, ignore path, `package-manager`, `base-image-address`, and Prettier args.
