# PSScriptAnalyzer

Official module: [dagger/PsScriptAnalyzer](https://github.com/dagger/PsScriptAnalyzer)

Install with `dagger mod install github.com/dagger/PsScriptAnalyzer`.

## What It Is

The PSScriptAnalyzer module checks PowerShell scripts with PSScriptAnalyzer.

## Use It For

- Checking PowerShell scripts
- Enforcing PowerShell style and safety rules
- Adding PowerShell lint checks to CI and Cloud

## Functions

| Function | Purpose |
| --- | --- |
| `check` | Run PSScriptAnalyzer on all PowerShell scripts in the workspace. |
| `base` | Base container with PSScriptAnalyzer installed. |
| `exclude` | PowerShell script paths to exclude from checking. |
| `include-extra-files` | Extra non-PowerShell file paths to include for scripts that need project context. |
| `version` | PSScriptAnalyzer version to install. |

## How to Read This API

Use `check` for workspace validation. Use `exclude` and `include-extra-files` to tune what the analyzer sees.

## Checks

Use `check` as the likely workspace check.
