# Pytest

Official module: [dagger/pytest](https://github.com/dagger/pytest)

Install with `dagger mod install github.com/dagger/pytest`.

## What It Is

The Pytest module runs Python tests with pytest and OpenTelemetry tracing. It can use the default Python project source or a custom container with Python and `uv` installed.

## Use It For

- Running Python tests
- Running test checks in CI and Cloud
- Sharing pytest setup across a team

## Functions

| Function | Purpose |
| --- | --- |
| `test` | Test a Python project with pytest and OpenTelemetry tracing. |
| `source` | Optional source directory containing the Python project. |
| `container` | Optional custom container with Python and `uv` installed. |

## How to Read This API

Start with `test`. Use `source` to point at the project directory. Use `container` when the default Python environment is not enough.
