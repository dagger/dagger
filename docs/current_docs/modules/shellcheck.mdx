# ShellCheck

Official module: [dagger/shellcheck](https://github.com/dagger/shellcheck)

Install with `dagger mod install github.com/dagger/shellcheck`.

## What It Is

The ShellCheck module finds shell scripts in a workspace and checks them with ShellCheck.

## Use It For

- Checking shell scripts
- Adding shell linting to CI and Cloud checks
- Sharing shell lint rules across a workspace

## Functions

| Function | Purpose |
| --- | --- |
| `check` | Run ShellCheck on all shell scripts in the workspace. |
| `scripts` | Find all `.sh` files in the workspace and return them as `Script` objects. |
| `base` | Base container with ShellCheck installed. |
| `exclude` | Shell script paths to exclude from checking. |

## How to Read This API

Use `check` for workspace validation. Use `scripts` when a workflow needs to inspect or compose per-script work.

## Checks

Use `check` as the likely workspace check.

## Settings

Document script discovery, `exclude`, ShellCheck version, and whether non-`.sh` shell files are included.

## Examples to Add

- Lint scripts in `scripts/`
- Lint all shell files in a repo
- Run as part of a workspace check suite
