# Vitest

Official module: [dagger/vitest](https://github.com/dagger/vitest)

Install with `dagger mod install github.com/dagger/vitest`.

## What It Is

The Vitest module runs Vitest tests for Vite and modern JavaScript or TypeScript projects. It uses a Node base image, package manager, and project source directory.

## Use It For

- Running Vitest tests
- Adding fast JS test checks
- Testing Vite-based workspaces

## Functions

| Function | Purpose |
| --- | --- |
| `test` | Execute the tests. |
| `list` | List the tests. |
| `source` | Source directory for the project. |
| `package-manager` | Package manager to use. |
| `base-image-address` | Base image to use. |

## How to Read This API

Use `test` for normal validation. Use `list` to inspect what Vitest would run.
