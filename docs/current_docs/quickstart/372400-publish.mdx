# Quickstart

## Package and publish the application

```shell
# package
dagger call \
  build-container --source=. --args=.

# check package contents
dagger call \
  build-container --source=. --args=.
  terminal

# publish
dagger call \
  build-container --source=. --args=. \
  publish --address=ttl.sh/my-container-$RANDOM
```
