# Quickstart

## Develop a custom function

So far, you've been calling individual Dagger Functions using the Dagger CLI. As you can see from the previous examples, as you chain more and more Dagger Functions together, using the CLI quickly becomes unwieldy.

A better approach is to develop your own custom Dagger Function, which offers greater control and flexibility. Creating your own Dagger Module enables you to transform your workflows into structured, discrete and composable units with clear inputs and outputs, with all the benefits of using a native programming language.
