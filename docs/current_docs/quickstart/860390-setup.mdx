# Quickstart

## Set up the example application

The example application used in this tutorial is a simple Go application.

Clone the application repository into your local development environment and make it the working directory:

```shell
git clone https://github.com/kpenfound/greetings-api.git hello-dagger
cd hello-dagger
```
