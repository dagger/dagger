# Generating Code

```shell
dagger generate
```

Runs all generators in your workspace. Each generator produces a [changeset](./changesets.mdx) — a diff you review before it touches your working tree.

## List generators

```shell
dagger generate -l
```

## Filter generators

```shell
dagger generate protobuf:*         # all generators from a module
dagger generate changelog:generate  # a single generator
```

## Verify in CI

Run the generators as checks. This fails the build if any generated file is out of date, without modifying your tree:

```yaml
# GitHub Actions
- run: dagger check --generate
```

See [Changesets](./changesets.mdx) for how reviewing and applying generated changes works locally.
