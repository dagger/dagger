# Welcome!

Welcome to Dagger's documentation site!

Dagger is a portable devkit for CI/CD pipelines that helps you build powerful CI/CD pipelines quickly, then run them anywhere.
Developed in the open by the creators of Docker.

In this page you'll find all the necessary information to navigate the docs your own way.
However, if you are looking for something specific, we recommend using the search bar.

If something is missing or you think could be improved, [ask for support](https://discord.gg/ufnyBtc8uY) or [create an issue](https://github.com/dagger/dagger/issues/new?template=documentation.yaml). Thanks for being part of the Dagger community!
