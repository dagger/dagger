The address (also known as "ref") of a container published as an OCI image.

Examples:
        - "alpine"
        - "index.docker.io/alpine"
        - "index.docker.io/alpine:latest"
        - "index.docker.io/alpine:latest@sha256deadbeefdeadbeefdeadbeef"
