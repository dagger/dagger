# Enumeration: DiffStatKind

The type of change for a diff stat entry.

## Enumeration Members

### Added

> **Added**: `"ADDED"`

A file or directory was added.

***

### Modified

> **Modified**: `"MODIFIED"`

A file was modified.

***

### Removed

> **Removed**: `"REMOVED"`

A file or directory was removed.

***

### Renamed

> **Renamed**: `"RENAMED"`

A file was renamed.
