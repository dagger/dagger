# Enumeration: ReturnType

Expected return type of an execution

## Enumeration Members

### Any

> **Any**: `"ANY"`

Any execution (exit codes 0-127 and 192-255)

***

### Failure

> **Failure**: `"FAILURE"`

A failed execution (exit codes 1-127 and 192-255)

***

### Success

> **Success**: `"SUCCESS"`

A successful execution (exit code 0)
