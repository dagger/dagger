# Type Alias: Platform

> **Platform** = `string` & `object`

The platform config OS and architecture in a Container.

The format is [os]/[platform]/[version] (e.g., "darwin/arm64/v7", "windows/amd64", "linux/arm64").

## Type Declaration

### \_\_Platform

> **\_\_Platform**: `never`
