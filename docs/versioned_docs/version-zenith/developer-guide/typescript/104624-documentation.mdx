## Function documentation

The following code snippet shows how to add document for a function in your module:

```typescript file=./snippets/documentation/function/index.ts
```

This documentation will be displayed for the `--help` flag of `dagger call`:

```shell
dagger call add --help
```

Here is an example of the result:

```shell
Compute the sum of two numbers.

Example usage: dagger call add --a=4 --b=5

Usage:
  dagger call add [flags]

EXPERIMENTAL:
  dagger call add is currently under development and may change in the future.

Flags:
      --a int   The first number to add (default 4)
      --b int   The second number to add
```
