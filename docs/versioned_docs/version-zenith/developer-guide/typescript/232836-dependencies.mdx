# Dependencies

<PartialExperimentalDocs />

To add a dependency to your module, you can use `dagger install`, as in the following example:

```sh
dagger install github.com/shykes/daggerverse/helloWorld@26f8ed9f1748ec8c9345281add850fd392441990
```

Dependent modules will be added to your code-generation routines, so you can access them from your own module's code, as shown below:

```typescript
@func()
async helloWorld(): Promise<string> {
    return await dag.helloWorld().message()
}
```
