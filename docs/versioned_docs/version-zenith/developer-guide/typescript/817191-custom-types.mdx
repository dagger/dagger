# Custom Types

<PartialExperimentalDocs />

TypeScript has multiple way to support complex data type, however, dagger supports it through fields (decorated with `@field()`) inside classes because `type` and `interface` keywords does not allow decorators.

This allows you to define alias using the field decorator option.
