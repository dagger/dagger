## Introduction

Welcome to Dagger, an integrated platform to orchestrate the delivery of applications from start to finish.

This tutorial introduces you to Dagger, a programmable tool that lets you replace your software project's artisanal scripts with a modern API and cross-language scripting engine.

It walks you, step by step, through the basics of using Dagger modules and functions with the Dagger CLI. It also shows you the basics of creating your own modules and functions.

## What you will learn

In this tutorial, you will learn how to:

- Install the Dagger CLI
- Call a Dagger function
- Understand Dagger function arguments and return values
- Chain Dagger functions together
- Use Dagger functions with containers
- Write your own Dagger function
- Use the Daggerverse

## Requirements

To use this tutorial, you only need [Docker](https://docs.docker.com/engine/install/) on your host system.
