## Introduction

Welcome to Dagger, an integrated platform to orchestrate the delivery of applications from start to finish.

This tutorial introduces you to Dagger, a programmable tool that lets you replace your software project's artisanal scripts with a modern API and cross-language scripting engine.

It walks you, step by step, through the different commands available in the Dagger CLI to work with Dagger modules. It shows you how to execute functions from modules and work with function outputs in new and powerful ways.

## What you will learn

In this tutorial, you will learn how to:

- Install the Dagger CLI
