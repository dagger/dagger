# Quickstart

## Create the example application

The example application used in this tutorial is a simple TypeScript application created with the Vue framework.

Create the application using the command below, answering "Yes" to all the prompts:

```shell
npm create vue@latest myapp
```

The example application will be created in the `myapp/` directory.
