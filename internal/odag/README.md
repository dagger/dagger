# ODAG

ODAG is Dagger's local trace ingestion, derivation, and web UI layer.

The most useful reference for current work is the data model:

- [DATA_MODEL.md](./DATA_MODEL.md)

That document describes:

- the primary telemetry substrate
- the derived entities surfaced by the API and UI
- the relationships between those entities
- current blurriness in the model and recommended simplifications
