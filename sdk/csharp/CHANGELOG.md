## v0.1.0 - 2025-11-25

### Added
- Complete C# SDK implementation
- Module runtime with attribute-based registration (`[Object]`, `[Function]`, `[Field]`, `[Interface]`)
- Interface support matching Python, TypeScript, and Go SDKs
- Roslyn analyzers for compile-time validation and code fixes
- C# docs translated to dagger function documentation
- NuGet package support for standalone client usage
- Example modules demonstrating SDK features
